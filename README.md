Golang app about setup a meetings between coworkers. In the meetings you can chat with other colleges and see their responses. It's load in the browser throught javascript.
Connection between the frontend and backend, I use graphql. The html content is full serverside rendered, nad then send to the browser.

#Usage: After clone the repo in the terminal execute:
# 1. docker compose build
# 2. docker compose up -d and then can start using it.
Have fun!
